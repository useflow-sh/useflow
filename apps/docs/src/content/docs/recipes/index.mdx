Recipes provide complete, production-ready examples for common flow patterns. Each recipe includes working code with detailed explanations.

## Available recipes

### [Onboarding Flow](/recipes/onboarding)
Complete user onboarding with form validation, persistence, and analytics tracking.

**Features:**
- Multi-step form with validation
- State persistence with localStorage
- Progress indicator
- Skip functionality
- Analytics integration

### [Checkout Flow](/recipes/checkout)
E-commerce checkout process from cart to confirmation.

**Features:**
- Cart management
- Address validation
- Payment integration
- Session persistence
- Order confirmation

### [Survey Flow](/recipes/survey)
Interactive survey or questionnaire with various question types.

**Features:**
- Rating questions
- Multiple choice
- Progress tracking
- Results visualization
- Anonymous responses
