useFlow is a declarative, type-safe library for building multi-step flows in React with **built-in persistence**. Whether you're building onboarding, checkout, surveys, or wizards, useFlow eliminates the complexity while providing exceptional developer experience.

## Quick example

Here's a complete onboarding flow in under 30 lines:

```tsx
import { defineFlow, Flow, useFlow } from "@useflow/react";

// 1. Define your flow structure
const onboardingFlow = defineFlow({
  id: "onboarding",
  start: "welcome",
  steps: {
    welcome: { next: "profile" },
    profile: { next: "complete" },
    complete: {}
  }
});

// 2. Create step components
function ProfileStep() {
  const { context, setContext, next, back } = useFlow();
  
  return (
    <div>
      <input 
        value={context.name}
        onChange={(e) => setContext({ name: e.target.value })}
      />
      <button onClick={back}>Back</button>
      <button onClick={next}>Continue</button>
    </div>
  );
}

// 3. Render with automatic step management
function App() {
  return (
    <Flow flow={onboardingFlow} initialContext={{ name: "" }}>
      {({ renderStep }) => renderStep({
        welcome: <WelcomeStep />,
        profile: <ProfileStep />,
        complete: <CompleteStep />
      })}
    </Flow>
  );
}
```

## Core concepts

### Flows
Define the structure and rules of your multi-step experience:

```tsx
const flow = defineFlow({
  id: "checkout",
  start: "cart",
  steps: {
    cart: { next: "shipping" },
    shipping: { next: "payment" },
    payment: { next: "confirm" },
    confirm: {}
  }
});
```

### Context
Shared state that flows through all steps:

```tsx
type CheckoutContext = {
  items: CartItem[];
  total: number;
  shippingAddress?: Address;
};

// Any step can access/update
const { context, setContext } = useFlow<CheckoutContext>();
```

### Type safety
TypeScript ensures you can't mess up:

```tsx
// Must provide ALL step components
renderStep({
  cart: <Cart />,
  shipping: <Shipping />,
  // âŒ Error: Missing 'payment' and 'confirm'
});

// Navigation is step-specific
const { next } = flow.useFlow({ step: "cart" });
next("shipping");  // âœ… Valid
next("confirm");   // âŒ Error: Can't skip to confirm from cart
```
